#  NutriCore Dashboard | Fitco AI

Prueba técnica se creara un Sistema de monitoreo metabólico con análisis de IA y actualizaciones en tiempo real

##  Stack de Librerías (Lista Oficial)
Se implementara las siguientes librerías para demostrar dominio técnico del ecosistema solicitado:

* **Backend:** `@nestjs/core`, `socket.io`, `mysql2`, `@nestjs/typeorm`, `@nestjs/config`.
* **Frontend:** `react`, `socket.io-client`, `react-hot-toast`, `lucide-react`.
* **IA:** `@google/generative-ai`
